¿En qué se diferencian Git y GitHub?

Git:

Git se trata un sistema de control de versiones que se utiliza para registrar y gestionar los cambios en archivos o proyectos. Funciona de manera local en una computadora y permite volver a versiones anteriores, comparar modificaciones y organizar el desarrollo de un proyecto. Por lo tanto, utilizamos Git como herramienta para controlar cambios de forma local.

GitHub:

Por otro lado, GitHub es una plataforma online que permite almacenar y compartir proyectos que utilizan Git. Funciona en la nube y facilita el trabajo colaborativo, además de permitir publicar proyectos y acceder a ellos desde cualquier lugar. Es decir, utilizamos GitHub como plataforma para almacenar y compartir proyectos de forma online.
